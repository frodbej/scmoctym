## Overview

This project introduced **scMOCTYM (single-cell Multi-Objective approach for Cell TYpe Markers)**, a multi-objective metaheuristic approach for generating cell type marker panels from single-cell transcriptomic data. This multi-objective optimization perspective allows to generate cell type marker panels by simultaneously minimizing the number of markers required and maximizing the panel's isolation power.
